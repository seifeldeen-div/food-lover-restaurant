# موقع مطعم Food Lover

موقع مطعم متكامل مع نظام تسجيل دخول، شراء منتجات، حجز طاولات، وسلة مشتريات.


### 1. نظام المصادقة
- ✅ صفحة تسجيل الدخول (`login.html`)
- ✅ صفحة التسجيل (`register.html`)
- ✅ تسجيل الخروج
- ✅ حماية الصفحات (يجب تسجيل الدخول للوصول)

### 2. صفحة المستخدم
- ✅ عرض معلومات المستخدم
- ✅ عرض إحصائيات (الطلبات، الحجوزات، المفضلة)
- ✅ روابط سريعة للصفحات المختلفة

### 3. صفحة المنتجات
- ✅ عرض جميع المنتجات
- ✅ تصفية حسب الفئة (إفطار، غداء، عشاء، حلويات)
- ✅ إضافة المنتجات للسلة
- ✅ إضافة/إزالة من المفضلة
- ✅ سلة المشتريات مع حساب الإجمالي
- ✅ إتمام الطلب
- ✅ عرض الطلبات السابقة

### 4. صفحة الحجز
- ✅ حجز طاولة (التاريخ، الوقت، عدد الأشخاص، نوع الطاولة)
- ✅ عرض الحجوزات السابقة
- ✅ إلغاء الحجز

### 5. سلة المشتريات
- ✅ إضافة/إزالة المنتجات
- ✅ تعديل الكمية
- ✅ حساب الإجمالي مع الضريبة (14%)
- ✅ عداد السلة في الهيدر

## الملفات

### الصفحات HTML
- `Restaurant web.html` - الصفحة الرئيسية
- `login.html` - تسجيل الدخول
- `register.html` - التسجيل
- `profile.html` - صفحة المستخدم
- `products.html` - صفحة المنتجات
- `booking.html` - صفحة الحجز

### ملفات CSS
- `Css/Restaurant.css` - التصميم الرئيسي
- `Css/auth.css` - تصميم صفحات المصادقة
- `Css/profile.css` - تصميم صفحة المستخدم
- `Css/products.css` - تصميم صفحة المنتجات
- `Css/booking.css` - تصميم صفحة الحجز

### ملفات JavaScript
- `js/app.js` - الوظائف الرئيسية المشتركة
- `js/auth.js` - نظام المصادقة
- `js/profile.js` - صفحة المستخدم
- `js/products.js` - صفحة المنتجات والسلة
- `js/booking.js` - صفحة الحجز
- `js/cart.js` - وظائف السلة

## كيفية الاستخدام

1. **التسجيل**: افتح `register.html` وأنشئ حساب جديد
2. **تسجيل الدخول**: استخدم `login.html` لتسجيل الدخول
3. **المنتجات**: اذهب إلى `products.html` لشراء المنتجات
4. **الحجز**: استخدم `booking.html` لحجز طاولة
5. **الحساب**: راجع معلوماتك في `profile.html`
